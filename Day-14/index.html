CSS

CSS:- Cascading Style Sheet.It is a stylesheet language that is used to describe the visual presentation of a web page written in HTML. HTML creates the structure of the page, while CSS adds styling to that structure.

We can access or use CSS with <style></style> tag

I have made all the div contents in red color using style tag 
        <style>
            div {
                color: red;
            }
        </style>

only the selected tags get the styling not all the tags in this case we used DIV tag and only div tag gets the styling
