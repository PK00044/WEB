Adding CSS to our HTML Site

    Three Ways
  1. Inline CSS
  2. Internal CSS
  3. External CSS
Inline CSS
This is used in an HTML page using <style> attributes
Internal CSS
It means to add a <style> tag and place HTML page's style within
External
It means to create a .cssfile and place all your styles inside that .css file
