CSS Transforms

    There are 2 types of Transforms:
  1. 2D Transform
  2. 3D Transform

2D Transform

This property allows the content to be restructured in ways like translate, rotate, scale or skew,etc. As it is 2D, therefore the content is transformed into x and y dimensions.

3D Transform

CSS transform 3D is a property that allows you to rotate, translate, and scale elements in a three-dimensional plane. It uses different methods to manipulate the position and orientation of the elements along the x, y, and z axes.
Transform origin is a point from where the box is rotatingg
