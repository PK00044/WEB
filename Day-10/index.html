The same goes with audio tags also just one different audio attribute is preload <audio src=".mp3" controls preload="" >:- Specifies if and how the audio should be loaded when the page loads("auto","metadata","none(default)")
  • None:- This is the default value.It indicates that the browser should not preload the audio file at all.The audio file will only start downloading when the user initates playback.
  • Metadata:- It only preloads the metadata of the audio file such as basic information,duration etc
  • Auto:- This instructs the browser to preload the entire audio file as much as possible.It starts to preload the audio file in background to get ready and can be available whenever the user is ready to play.

SVG(Media)

Scalable Vector Graphics(SVG) has become an indispensable part if modern web development.SVG enables developers to create high-quality,Scalable Graphics that look crisp at any size or resolution.

SVG stands for Scalable Vector Graphics. Unlike raster images like PNGs or JPGs, SVGs are not pixel-based. They're composed of vectorsâ€”mathematical formulas that describe shapes, paths, and fills. This means SVGs can be resized without losing quality.

Why Use SVG?

Scalability

SVG images can be scaled indefinitely without losing quality, which is ideal for responsive web design.

File Size

SVG files are often smaller than their raster counterparts, especially for simple shapes and icons.

Flexibility

SVGs can be styled, animated, and manipulated using CSS and JavaScript.

How to Embed SVG in HTML

SVG can be embedded in HTML in several ways:
  1. Inline SVG: Directly writing the SVG XML code within HTML.
  2. Using an <img> tag: Point the src attribute to an SVG file.
  3. Using CSS: Setting SVG as a background image in a CSS file.

Inline SVG Example

<svg height="100" width="100"> <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /></svg>

<img> Tag Example

<img src=".svg" alt="Sample SVG">
You have to create a file of .svg and store the code of the image and mention xmlnsto work or else it won't work.

CSS Background Example

.background { background-image: url('image.svg'); }

SVG Attributes

SVG comes with a set of attributes to control its behavior:
  1. width and height: To set the dimensions.
  2. viewBox: To set the coordinate system.
  3. fill and stroke: To set the colors.

Creating Complex Shapes

SVG can also be used to create more complex shapes like polygons, lines, and text.

SVG offers a powerful way to add scalable and interactive graphics to your HTML documents. Its compatibility with CSS and JavaScript makes it a versatile choice for modern web development. Whether you're creating simple icons or intricate illustrations, SVG has got you covered.

IFRAME

iFrames, or Inline Frames, are an integral part of modern web development. They allow you to embed another HTML page within your current page.We'll delve into the utility of iFrames, their attributes, and some use-cases.

An iFrame is an HTML element that enables an inline frame for the embedding of external content. Essentially, you can load another web page within a designated area of your current webpage.

Why Use iFrames?

iFrames offer a variety of use-cases:
  • Content Isolation: iFrames allow you to isolate third-party content, which can improve security.
  • Modularity: Easily embed external plugins, widgets, or content.
  • Resource Separation: Content within an iFrame can load separately from the rest of the page.

Basic Syntax

<iframe src="URL" width="width" height="height"></iframe>

Attributes of iFrame

Several attributes can enhance the functionality of an iFrame:
  1. src: Specifies the URL of the page to embed.
  2. height and width: Define the dimensions.
  3. frameborder: Indicates whether to display a border.
  4. scrolling: Controls the scrollbars.
  5. name: For targeting the iFrame in
    6. JavaScript

A Practical Example

<iframe src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

Embedding Google Maps

<iframe src="https://maps.google.com/maps?q=LOCATION&output=embed" frameborder="0"></iframe>

iFrames offers a convenient way to embed external content into your web pages. Their flexibility and ease of use make them an invaluable tool for modern web development.
