CSS Float and Clear

Just like how “wrap through” works in MS Word, the float property allows the balance between text and the image shown beside it.

Float is used to float the image / any text to either left or right and clear is used to tell that the particular thing which is floating should to appear on left or right or both(L or R)

But not recommened to use a lot when we have flexbox
