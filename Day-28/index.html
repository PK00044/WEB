CSS Positioning

The CSS positions allow you to precisely control the placement of an element on the web page.

It helps to determine how elements are placed inside the container element and how they interact with the other elements on the page.

There are various types of position property values, such as:

position: relative means that the one who had his own place and we can now apply top bottom z-index etc like properties.

position : """absolute""" when we apply any of the properties say like
"""top : 0px """, it refers to the nearest positioned element / ancestor.
position : fixed makes ur element fixed at one position through out the page.

Static(Default)

The elements are positioned according to the normal flow of the document.

Syntax: 

        selector {
            position: static;
      }
Example:- 

        <head>
        <style>
            img {
                position: static;
            }
        </style>
    </head>
    <body>
        <p id="p1">Lorem ipsum dolor sit amet.

        <p id="p2">Lorem ipsum dolor sit amet.

        <p id="p3">Lorem ipsum dolor sit amet.

        <p id="p4">Lorem ipsum dolor sit amet.

    </body>
    </html>

Relative

Elements are positioned relative to the normal position in the document.

You can use the top, right, bottom, and left properties to move the element from its original position.

Syntax:- selector { position: relative; } Example:- 

            <head>
                <style>
                    img {
                        position: relative;
                        left: 100px;
                        top: 50px
                    }
                </style>
            </head>
            
            <body>
                <p id="p1">Lorem ipsum dolor sit amet.

                <p id="p2">Lorem ipsum dolor sit amet.

                <img src="./logo.png" alt="logo">
                <p id="p3">Lorem ipsum dolor sit amet.

                <p id="p4">Lorem ipsum dolor sit amet.

            </body>
            </html>

Absolute

Elements are positioned relative to the closest positioned ancestor (parent), which means we need to have a parent element with a positioning other than 'static'.

Note: An absolutely positioned element is removed from the normal flow.

Example:- 

        <head>
            <style>
                #about{
                    position: relative;
                }
                .logo{
                    position: absolute;
                    right: 10px;
                    top: 10px;
                }
            </style>
        </head>
        <body>
            <h1>CodeWithHarry
            <div class="about">
                <p>Developer

                <p>founder of CodeWithHarry

                <img class="logo" src="./logo.png" alt="logo">
            </div>
        </body>
        </html>
OutPut:-

CodeWithHarry

Fixed

Elements are positioned relative to the viewport (screen) and do not move when the page is scrolled.

This is useful for creating elements like fixed headers or footers.

Example:- 

    <head>
        <style>
                h1{
                    position: fixed;
                    top: 10px;
                    right: 20px;
                }
        </style>
        </head>
    <body>
            
CodeWithHarry

    </body>
    </html>
OutPut:-

CodeWithHarry

Here, the image position will be fixed.

Float

The float property is used to shift an element to the left or right within its containing element.

Sticky

Position sticky is a hybrid between 'relative' and 'fixed'.

It allows an element to become "stuck" to the top or bottom of its container when scrolling, but it behaves like relative positioning within the container until it reaches a specified offset.

Example:- 

        <head>
            <style>
                h1{
                    position: sticky;
                    top: 10px;
                    right: 20px;
                }
            </style>
        </head>
        <body>
            <h1>CodeWithHarry
        </body>
        </html>
OutPut:-

CodeWithHarry
